AI-BOS is a cloud-based platform that combines CRM, task management, document intelligence, and AI assistants to help companies run operations.

# AI-BOS Platform

## Architecture
Microservice-based system using NestJS and Docker.

## Services
- Auth
- AI
- CRM
- Task
- Document

## Tech Stack
- Node.js (NestJS)
- PostgreSQL
- Redis
- Docker
- GitHub Actions

## Run Locally
docker compose up --build

## CI/CD
GitHub Actions builds and validates all services.
